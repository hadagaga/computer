IP（Internet Protocol，网际协议）负责为主机和网络接口提供地址，并让路由设备能够把数据报逐跳转发到目标网络。IP 关注尽力而为的交付，不保证数据一定抵达、按序抵达或只抵达一次。

## 核心问题

1. IP 地址如何标识网络与主机？
2. 路由器如何决定数据报的下一跳？
3. 为什么可靠传输不是 IP 自身的职责？
